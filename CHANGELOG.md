# Changelog

All notable updates to this project are listed here in a simple, beginner-friendly format.

## [0.1.0] - 2026-03-19

Initial portfolio-ready version.

### Added

- public landing page
- shared layout with header and sidebar
- Clerk authentication
- protected routes for dashboard, inventory, and prep
- database-backed inventory CRUD
- inventory search and filters
- low-stock alerts and status badges
- dashboard summaries
- database-backed prep task CRUD
- Zod validation for forms
- automated tests with Vitest and Testing Library
- GitHub Actions CI
- release, screenshot, deploy, and QA support docs
- accessibility, semantic HTML, and mobile polish

### Notes

- This version is intentionally beginner-friendly and focused on full-stack fundamentals.
- Future improvements can build on this foundation without changing the core portfolio scope.
